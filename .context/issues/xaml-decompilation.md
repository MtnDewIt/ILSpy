## XAML DECOMPILATION ISSUES

---

Below are various examples of XAML decompilations that are non functional by default in the output code. The first chunk is the code generated by ILSpy, the second chunk is the code that is actually functional.

### EXAMPLE 1

#### ORIGINAL
```xml
<corinth:CachedRotatingRing>
  <UIElement.Visibility>
    <Binding ElementName="userControl">
      <Ctor xmlns="https://github.com/icsharpcode/ILSpy">IsLoadingBitmap</Ctor>
      <Binding.Converter>
        <corinth:VisibilityConverterExtension corinth:VisibilityConverterExtension.ConversionMode="HiddenWhenFalse" />
      </Binding.Converter>
    </Binding>
  </UIElement.Visibility>
</corinth:CachedRotatingRing>
```

#### FIXED
```xml
<corinth:CachedRotatingRing Visibility="{Binding IsLoadingBitmap, ElementName=userControl, Converter={corinth:VisibilityConverter ConversionMode=HiddenWhenFalse}}" />
```

### EXAMPLE 2

#### ORIGINAL
```xml
<Rectangle>
  <FrameworkElement.Width>
    <Binding ConverterParameter="14.0">
      <Ctor xmlns="https://github.com/icsharpcode/ILSpy">DisplayNestLevel</Ctor>
      <Binding.Converter>
        <corinth:MathConverterExtension corinth:MathConverterExtension.MathMode="Multiply" />
      </Binding.Converter>
    </Binding>
  </FrameworkElement.Width>
</Rectangle>
<ToggleButton VerticalAlignment="Stretch" Width="14" Style="{StaticResource {x:Staticcorinth:ButtonResources.ExpanderToggleButtonStyleKey}}" IsChecked="{Binding IsExpanded}">
  <UIElement.Visibility>
    <Binding>
      <Ctor xmlns="https://github.com/icsharpcode/ILSpy">HasChildren</Ctor>
      <Binding.Converter>
        <corinth:VisibilityConverterExtension corinth:VisibilityConverterExtension.ConversionMode="HiddenWhenFalse" />
      </Binding.Converter>
    </Binding>
  </UIElement.Visibility>
</ToggleButton>
```

#### FIXED
```xml
<Rectangle Width="{Binding DisplayNestLevel, ConverterParameter=14.0, Converter={corinth:MathConverter MathMode=Multiply}}" />
<ToggleButton VerticalAlignment="Stretch" Width="14" Style="{StaticResource {x:Static corinth:ButtonResources.ExpanderToggleButtonStyleKey}}" IsChecked="{Binding IsExpanded}" Visibility="{Binding HasChildren, Converter={corinth:VisibilityConverter ConversionMode=HiddenWhenFalse}}" />
```

Ideally, we would want to have the decompiler output the fixed versions of the XAML instead of the original versions, since the original versions are non functional by default and require manual fixing to be usable, whcih is greate when this issue may occur hundreds of times across a given assembly.

To address these issue, the handling for decompiling the BAML data pulled from assemblies can be found in the `ICSharpCode.BamlDecompiler` and `ILSpy.BamlDecompiler` projects.
